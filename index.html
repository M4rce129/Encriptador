Manual de Uso – Herramienta de Encriptado
Esta herramienta permite codificar y decodificar mensajes usando un sistema personalizado de encriptación.
Además, cuenta con autenticación de usuarios y utilidades para facilitar su uso desde la consola. El programa se maneja a través de palabras clave o comandos que el usuario debe ingresar en la consola.
Las entradas de información en la consola se indican con << y las salidas con >>.
Inicio de Sesión
  • 1. Ingresa tu nombre de usuario cuando el programa lo solicite.
  • 2. Ingresa la LLAVE de acceso. Si los datos son correctos, accederás al menú principal.
Comandos Disponibles
  • HELP – Muestra la lista de comandos disponibles y una breve descripción de cada uno.
  • TEXT – Convierte un texto en un código encriptado. Escribe TEXT y luego ingresa el texto que deseas codificar.
  • CODE – Convierte un código encriptado de vuelta a texto. Escribe CODE y luego ingresa la secuencia de números generada por la opción TEXT.
  • EXIT – Sale del programa de forma segura.
  • CLEAR – Limpia la pantalla de la consola para una mejor visualización.
  • INFO – Muestra información sobre el usuario actual y detalles del programa.
  • MANUAL – Abre este manual o el enlace al tutorial en video.
Consejos de Uso
  • Si ingresas un carácter no soportado, será marcado como -1 o ? en la decodificación.
  • Usa CLEAR para limpiar la pantalla si la consola se ve saturada.
  • Si necesitas ayuda en cualquier momento, escribe HELP.
Errores Comunes
  • Usuario o contraseña incorrectos: El programa terminará si los datos no coinciden.
  • Interrupción con Ctrl+C: El programa detecta la interrupción y muestra un mensaje amigable.
  • Caracteres no válidos: Se mostrarán como ? al decodificar.
